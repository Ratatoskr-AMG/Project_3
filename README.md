# DoHeCo
Это каталог героев DOTA2, который обеспечивает доступ к данным, предоставленным https://www.opendota.com/. Каталог ползволяет Вам сравнивать героев друг с другом, фильтровать по ролям, эффективности и т.п.

This is a catalog of DOTA2 heroes, which provides access to the data supplied with https://www.opendota.com/. The catalog allows you to compare heroes with each other, filter by roles, effectiveness, etc.
